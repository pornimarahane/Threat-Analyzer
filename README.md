# Apexia-IQ-Project

Threat Analyzer and Simulator  
Tech Stack: Vue.js, Vuetify, MongoDB, REST APIs/n  
Developed a threat detection tool that analyzes URLs or uploaded files to classify them as threat or non-threat using API. Included an interactive simulator where users can practice basic cyber-attack scenarios and test their knowledge through quiz-based activities.

Installation and setup:  
1. Clone the repository  
   git clone https://github.com/pornimarahane/Threat-Analyzer.git  
   cd Threat-Analyzer  

2. Install dependencies  
   npm install  
   cd frontend && npm install  
   cd ../backend && npm install  

3. Running the Project  
   From root folder: npm run both  
   OR separately:  
   Backend: npm run backend  
   Frontend: npm run frontend
